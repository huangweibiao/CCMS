## 🔍 问题排查指南

### 常见启动问题

#### 问题1：依赖安装失败
**错误信息**：`npm ERR! code ERESOLVE`
**解决方案**：
```bash
# 清除npm缓存
npm cache clean --force

# 重新安装依赖
npm install
```

#### 问题2：端口冲突
**错误信息**：`Error: listen EADDRINUSE: address already in use`
**解决方案**：
1. **停止占用端口的进程**：
   ```bash
   # 查找占用端口的进程
   netstat -ano | findstr :3000
   # 或者使用端口查看工具
   ```

2. **修改配置文件**：
   ```bash
   # 编辑vite.config.js，更改端口
   server: {
     port: 3001  # 更改端口号
   }
   ```

#### 问题3：API连接失败
**错误信息**：`Network Error` 或 `404 Not Found`
**解决方案**：
1. **检查后端服务状态**：
   ```bash
   # 查看后端是否正常运行
   curl http://localhost:8080/api/health
   ```

2. **检查API代理配置**：
   ```bash
   # 检查vite.config.js中的代理配置
   server: {
     proxy: {
       '/api/finance': {
         target: 'http://localhost:8080',
         changeOrigin: true
       }
     }
   }
   ```

3. **检查后端端口配置**：
   ```bash
   # 查看application.yml中的端口配置
   server:
     port: 8080
   ```

#### 问题4：页面空白或加载异常
**错误信息**：白屏或页面无响应
**解决方案**：
1. **检查浏览器控制台错误**：
   - F12 打开开发者工具
   - 查看Console错误信息

2. **检查网络请求**：
   - 在Network面板查看API请求状态
   - 验证请求参数是否正确

3. **清除浏览器缓存**：
   - Ctrl + Shift + Delete (清除缓存)
   - Ctrl + F5 (强制刷新页面)
